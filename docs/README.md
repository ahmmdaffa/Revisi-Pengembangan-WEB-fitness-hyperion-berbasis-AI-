# Project Documentation

Folder ini berisi dokumen pendukung untuk presentasi dan penilaian project HYPEFIT.

## Isi Folder

- `functional-prototype/`  
  Laporan functional prototype, screenshot aplikasi, dan versi PDF/DOCX.

- `presentation/`  
  Outline materi presentasi per slide.

Dokumen di folder ini tidak dibutuhkan untuk menjalankan aplikasi, tetapi dipertahankan sebagai artefak capstone.
